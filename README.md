# 🌌 The Specialist Portfolio | Mission Control

A high-tech, immersive developer portfolio designed with a "Cyber-Specialist" aesthetic. This project features a dual-mode interface: a familiar Google-inspired search gateway that transitions into a sophisticated, holographic mission control dashboard.

![Portfolio Preview](https://via.placeholder.com/1200x600/020617/3b82f6?text=MISSION+CONTROL+V4.2)

## 🚀 Experience the Interface

The portfolio is built to mimic a high-security data vault, providing a unique storytelling experience for recruiters and collaborators.

- **Search Gateway**: A minimalist "Google Search" home screen that handles real-time data indexing.
- **Dossier Inspection**: An RPG-style "About" page featuring holographic scanners and live data feeds.
- **Ability Matrix**: A dynamic skills visualization with rotating neural orbits and proficiency sync tracking.
- **Mission Archive**: A 3D-inspired project showcase using wireframe carousels and tactical briefings.
- **Hall of Fame**: An achievement tracker with global level progression and verified logs.
- **Comms Hub**: A secure communication terminal with satellite link simulation and encrypted transmission.

## 🛠 Tech Stack

- **Core**: React 19 + Vite
- **Styling**: Tailwind CSS 4 (Blue/Cyan/Navy Palette)
- **Animation**: Framer motion (Layout transitions, holographic effects)
- **Icons**: Font Awesome (Stable Integration)
- **Typography**: Outfit (Modern Geometric Sans)

## 🏗 Project Structure

```bash
src/
├── components/      # Reusable UI Modules (Terminal, SideMenu, SearchBox)
├── data/            # Centralized Content Repository (Projects, Skills, Bio)
├── pages/           # High-Tech Interface Layouts (About, Skills, Mission Archive)
├── assets/          # Static Media & Specialist Assets
└── index.css        # Global Design System & Holographic Tokens
```

## ⚙️ Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/574-jayeshSh/portfolio.git
cd portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Launch Mission Control
```bash
npm run dev
```

The interface will be accessible at `http://localhost:5173/`.
